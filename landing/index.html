App herunterladen

Wähle dein Betriebssystem. Keine Installation von Zusatzsoftware nötig – einfach starten.

Empfohlen
🪟 Windows

Windows 10 / 11. Öffnet sich als eigenes Programmfenster mit dem Waschbär-Symbol.

⬇ edeka-promo-tool-windows.zip
🍎 macOS

macOS-App. Beim ersten Start: Rechtsklick → „Öffnen".

⬇ edeka-promo-tool-mac.zip
🐧 Linux

Paket entpacken und ./install.sh ausführen.

⬇ edeka-promo-tool-linux.tar.gz

Hinweis (Windows): Falls eine Warnung erscheint → „Weitere Informationen" → „Trotzdem ausführen". Die App läuft lokal und sicher auf deinem Rechner.
